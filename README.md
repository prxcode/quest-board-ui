# Quest Board UI
A single-page interactive Quest Card built with HTML, CSS, and JavaScript for the Adventurer's Guild.

> **Note:** This repository exists purely as a functional demonstration of vanilla DOM manipulation and basic CSS styling as required for GDG Application.


## Features
- Dark fantasy themed quest card with quest details (reward, difficulty, deadline)
- "Accept Quest" button that changes text to "Quest Accepted!", turns gray, and becomes disabled on click
- Responsive layout for mobile and desktop
- Google Fonts (Cinzel + Inter) for a polished look

## Tech Stack
- HTML5
- CSS3 (vanilla)
- JavaScript (vanilla)
